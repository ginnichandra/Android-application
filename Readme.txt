When your app starts it will show a login activity. 
If you're not a subscribed user, then you need to get registered by clicking register button.
After a user is registered it redirectes you to login page.
When successfully logined the app will show a list view activity.
It will be a list of city names. You have an option to add a city, to add a city the user must enter a city name and 
latitude and longitude values.
You can see that the city is updated in the list
